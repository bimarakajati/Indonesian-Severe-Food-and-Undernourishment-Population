# Indonesian-Severe-Food-and-Undernourishment-Population

**World Development Indicators** (WDI) berisikan metrik-metrik untuk menunjukkan performa perkembangan pada suatu negara dengan satuan waktu yaitu tahun.

Data statistik WDI ini sangat berguna untuk mengetahui alasan kenapa suatu negara dikatakan sudah maju ataupun masih merupakan negara berkembang.

Adapun World Development Indicator mencakup sektor-sektor yang terlibat di bawah ini, yaitu:
1.	Health
2.	Education
3.	Environment
4.	Infrastructure
5.	Economics and Finance
6.	Social
